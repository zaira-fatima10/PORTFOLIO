YouTube Summarizer Chrome Extension

GitHub Link

Overview: The YouTube Summarizer Extension is a powerful tool designed to enhance your YouTube viewing experience by providing concise summaries of video content. This browser extension leverages advanced natural language processing (NLP) models to extract and summarize the transcript of YouTube videos, making it easier for users to grasp the key points without watching the entire video.

Key Features

Video Summarization: Automatically generates a summary of the YouTube video's transcript, highlighting the main points and key takeaways.
Multilingual Translation: Translates the summarized text into multiple languages, including Hindi, Gujarati, and Urdu, making the content accessible to a broader audience.
Text-to-Speech: Converts the summarized text into speech, allowing users to listen to the content summary.
Transcript Download: Provides an option to download the summarized transcript as a text file for offline reading.

Technology Stack

Frontend: HTML, CSS, JavaScript for creating the user interface.
Backend: Flask for handling API requests and processing the YouTube transcript.
NLP Models: Hugging Face Transformers for summarization and translation tasks, Google Text-to-Speech (gTTS) for generating audio.

How It Works

Input YouTube URL: Users input the URL of the YouTube video they want to summarize.
Fetch Transcript: The extension retrieves the video transcript using the YouTube Transcript API.
Generate Summary: The transcript is processed through a summarization model to create a concise summary.
Translation: Users can translate the summary into Hindi, Gujarati, or Urdu.
Text-to-Speech: The summarized text can be converted into audio.
Download Transcript: Users can download the summarized transcript for offline use.

Use Cases

Quick Information Access: Quickly understand the key points of lengthy YouTube videos without watching the entire content.
Language Accessibility: Non-native English speakers can access video content in their preferred language.
Audio Summaries: Ideal for users who prefer listening to summaries instead of reading.

Installation and Usage

Install the Extension: Add the YouTube Summarizer Extension to your browser.
Enter Video URL: Open the extension and enter the URL of the YouTube video.
Summarize: Click on the "Summarize" button to generate the video summary.
Translate and Listen: Use the translation and text-to-speech features as needed.
Download: Save the summarized transcript for future reference.

This project showcases the integration of modern NLP techniques with practical applications, demonstrating my skills in software development, natural language processing, and user interface design.
